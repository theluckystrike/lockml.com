# LockML -- Open Source ML Model Comparison Table

**[-> Use LockML (live tool)](https://lockml.com/)**

LockML is a free interactive comparison table for 30+ open-source machine learning models. Filter by parameter count, license type, and use case tags. Sort by MMLU benchmark scores, release dates, and model size. Find the right open-source LLM for your project with real benchmark data from official papers and the Open LLM Leaderboard.

## Features

- 30+ open-source ML models compared side-by-side
- Filter by parameter count with interactive slider
- Filter by license: Apache 2.0, MIT, or restrictive/custom
- Use case tags: chat, code, RAG, multilingual, edge deployment
- Sortable columns: model name, org, parameters, license, release date, MMLU
- Real benchmark data from official papers and Open LLM Leaderboard
- License compatibility checker for commercial use
- 100% client-side -- your data never leaves your browser
- Open source -- inspect the code yourself

## Tech Stack

- Vanilla JavaScript (no frameworks, no build step)
- Static HTML hosted on GitHub Pages
- Cloudflare DNS + SSL
- Zero dependencies, zero tracking, zero cookies
