# ML Model Comparison Tool — Compare 30+ Machine Learning Models Side by Side

**[Compare Models →](https://lockml.com)** | [About](https://lockml.com/about.html) | [Blog](https://lockml.com/blog/)

ML Model Comparison Tool lets you compare 30+ machine learning models across parameters, performance benchmarks, license type, and use cases. Filter by model size, architecture, and license. Sort by any column. Expand any model card for deep-dive details. Whether you're choosing between BERT variants, comparing LLMs, or evaluating open-source vs. proprietary options, this tool puts the data you need in one place.

## Features

- **30+ ML models catalogued** — covering LLMs, vision models, embedding models, and more
- **Filter by parameters, license, and architecture** — narrow down your options fast
- **Sortable comparison table** — rank models by any metric that matters to you
- **Expandable model cards** — detailed specs, benchmarks, and use case recommendations
- **Always up-to-date** — new models added as they're released

## How It Works

Open the comparison dashboard and use the filter controls to narrow models by type, parameter count, license, or provider. Click any column header to sort. Click a model row to expand its full card with architecture details, benchmark scores, training data info, and recommended use cases. All data is loaded client-side for instant filtering with no server round-trips.

## Built With

- Client-side only — your data never leaves your browser
