# LockML

> Free open-source ML model comparison table with real MMLU benchmark data, license filters, and use-case tagging.

**[Use LockML live](https://lockml.com/)**

LockML is a free interactive comparison table for 30+ open-source machine learning models. Filter by parameter count, license type, and use-case tags. Sort by MMLU benchmark scores, release dates, and model size. Find the right open-source LLM for your project with real benchmark data pulled from official papers and the Open LLM Leaderboard — no sign-up required.

## Features

- 30+ open-source ML models compared side-by-side
- Filter by parameter count using an interactive slider
- Filter by license: Apache 2.0, MIT, or restrictive/custom
- Use-case tags: chat, code, RAG, multilingual, edge deployment
- Sortable columns: model name, organization, parameters, license, release date, MMLU
- Real benchmark data from official papers and the Open LLM Leaderboard
- License compatibility checker for commercial use
- Direct links to Hugging Face model cards and weight downloads
- 100% client-side — no data leaves your browser
- MIT licensed
- No signup, no tracking

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- No build step
- No external dependencies (except Google Fonts on some pages)
- Hosted on GitHub Pages with Cloudflare CDN
