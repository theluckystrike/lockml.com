About LockML

What is LockML?

LockML is a free, interactive comparison table for open-source machine learning models. It provides real benchmark data, parameter counts, licensing information, and use case tags for 30+ models — all in one filterable, sortable view.

The ML landscape moves fast. New open-source models ship every week, each with different strengths, licenses, and deployment requirements. LockML cuts through the noise and gives you the data you need to make an informed choice.

Data Sources

Model data is compiled from official announcements, Hugging Face model cards, and published benchmark papers. MMLU scores come from the original papers or the Open LLM Leaderboard where available. We update the data regularly as new models are released.
