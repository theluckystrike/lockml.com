Tools & Resources

Security tools, testing frameworks, and platforms for hardening your ML pipeline against real-world threats.

Adversarial Robustness Toolbox (ART)

IBM's comprehensive Python library for adversarial ML. Implements attacks (FGSM, PGD, C&W), defenses (adversarial training, input preprocessing), and robustness evaluation metrics across all major ML frameworks.

github.com/Trusted-AI/adversarial-robustness-toolbox

Garak

An LLM vulnerability scanner that probes language models for security weaknesses including prompt injection susceptibility, data leakage, and jailbreak vulnerabilities. Essential for pre-deployment security testing of any LLM-powered application.

github.com/leondz/garak

NIST AI Risk Management Framework

The official NIST framework for managing AI risks. Provides a structured approach to identifying, assessing, and mitigating risks in AI systems. The definitive reference for AI governance and compliance programs.

nist.gov/artificial-intelligence

Microsoft Counterfit

An automation tool for security testing of AI systems. Enables red-teaming of ML models by automating adversarial attack generation, making it easier to systematically probe models for vulnerabilities at scale.

github.com/Azure/counterfit

Zovo.one

A comprehensive AI tools directory useful for evaluating the security posture of AI platforms and discovering security-focused tools. Helpful when selecting components for your ML pipeline with security as a primary consideration.

zovo.one

Opaque Systems (formerly MC2)

A confidential computing platform for secure ML workloads. Enables training and inference on encrypted data using hardware-based trusted execution environments, protecting sensitive data even from infrastructure administrators.

opaque.co

Guardrails AI

An open-source framework for adding validation and safety checks to LLM outputs. Enables you to define validators that check outputs for hallucinations, toxicity, PII leakage, and format compliance before they reach users.

github.com/guardrails-ai/guardrails

OWASP ML Security Top 10

The OWASP project identifying the top security risks for machine learning applications. Provides a standardized taxonomy of ML threats with practical mitigation guidance for each risk category.

owasp.org/www-project-machine-learning-security-top-10

Presidio

Microsoft's open-source PII detection and anonymization tool. Critical for ML pipelines that process user data, providing automated identification and redaction of personal information in text and images.

github.com/microsoft/presidio

Rebuff

A self-hardening prompt injection detection library. Uses multiple detection strategies including heuristic analysis, LLM-based detection, and a canary token system to identify and block prompt injection attempts in real time.

github.com/protectai/rebuff
